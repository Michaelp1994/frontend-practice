# Frontend Practice

The project uses Vite as a development server. To demo it, run the following terminal commands:

- `npm install`
- `npm run start`

You can then visit the app in-browser; it defaults to http://localhost:3000.

Stack used:

- Typescript
- Figma
- Vite
- React V18
- React Router
- Redux
- Recharts
- Styled Components
